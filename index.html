MEM MCP Package Repository

AI-powered version control tool - memory layer for AI coding agents

Homebrew (macOS/Linux)

brew install --formula https://IRONICBo.github.io/mem-mcp-server/homebrew/Formula/mem-mcp.rb

APT (Debian/Ubuntu)

echo "deb [trusted=yes] https://IRONICBo.github.io/mem-mcp-server/apt stable main" | sudo tee /etc/apt/sources.list.d/mem.list
sudo apt update
sudo apt install mem

YUM/DNF (Fedora/RHEL)

sudo curl -o /etc/yum.repos.d/mem.repo https://IRONICBo.github.io/mem-mcp-server/yum/mem.repo
sudo dnf install mem
